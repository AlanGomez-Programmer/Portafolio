## 🎨 Paleta de Colores

El diseño utiliza una paleta moderna de **tonos azules oscuros con acentos cian**, inspirada en interfaces modernas y profesionales:

| Color | Código | Uso |
|-------|--------|-----|
| **Fondo Principal** | `#0d1117` | Fondo base de toda la página |
| **Fondo Secundario** | `#161b22` | Secciones y contenedores |
| **Azul Primario** | `#3b82f6` | Acentos y detalles destacados |
| **Cian/Azul Cielo** | `#38bdf8` | Hover effects y bordes |
| **Texto Principal** | `#ffffff` | Títulos y encabezados |
| **Texto Secundario** | `#c9d1d9` | Párrafos y contenido |
| **Bordes** | `rgba(56, 189, 248, 0.3)` | Bordes sutiles con transparencia |
